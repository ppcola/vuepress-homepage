## 传智书城

<ProjectCard>

  **JavaWeb传智书城程序设计**
  
  JavaWeb传智书城是一套基于JavaWeb技术实现的在线书城系统，主要功能包括用户登录注册、书籍查询、购物车管理、订单管理、支付和评价等。该系统采用了B/S架构，基于SpringBoot框架、MyBatis-Plus数据库ORM框架和Vue.js前端框架实现。系统的后端通过Spring Security框架实现了用户的权限控制，保证了系统的安全性。
